Helmet detector About Our project

The HelmetDetection project is a complete end-to-end machine learning classification project that detects helmets in images. The project's main objective was to create and deploy a machine learning model using images scraped from the internet. The project provides hands-on experience using popular tools such as Roboflow, Hugging Face, and Flask.

To generate the dataset, the project used SerpApi from the Googlesearch module to scrape images from the internet. A Python script called Image_scrap.py was used to download the images, which were saved in the SerpApi_images folder. The dataset was then labeled using Roboflow, which helped to efficiently and accurately label the images.

After labeling, the dataset was pre-processed by dividing it into a 70-20-10 train-val-test split and resized to 224x224 dimensions. Augmentation was also applied to generalize the model. The pre-processed dataset was uploaded to Hugging Face using the Helmet_model.ipynb file.

The model was then trained using ViTForImageClassification from the transformers module, achieving a 95% training accuracy with 0.25 validation loss. The model was deployed in Hugging Face using the Helmet_model.ipynb file and is available at https://huggingface.co/pzalavad/AiCampHelmet for use in various platforms.

Finally, a simple website was created, and the model was consumed using Flask to showcase its effectiveness in detecting helmets in images. This project highlights the power and practicality of machine learning in real-world applications, and the use of popular tools such as Roboflow, Hugging Face, and Flask demonstrates the versatility of machine learning.
